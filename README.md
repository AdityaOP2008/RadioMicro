# RadioMicro
__A discord radio bot to play radio on discord. Installed on discord.js version 13.7.0__

**Start**
```md
node src/index.js
node .
```

**Requirements**
```md
discord.js v13.6.0 or v13.7.0
Node above v16.6.0 and below 17
```

**Features**<br>
• 24/7 Play Mode<br>
• Lofi & Piano System<br>
• 11+ Radio Stations<br>
